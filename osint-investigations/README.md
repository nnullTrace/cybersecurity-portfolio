# 🟣 OSINT Investigations
This section contains Open Source Intelligence (OSINT) investigations focused on identifying patterns, behaviors, and potential risks using publicly available information.


## 🎯 Objective
* Analyze real-world online activity
* Identify behavioral patterns and inconsistencies
* Detect potential risks (scams, misinformation, irregular activity)
* Correlate data across multiple platforms


## 🔍 Methodology
* Keyword-based search (multi-language when needed)
* Social media analysis
* Cross-platform correlation
* Pattern & behavioral analysis
* Verification using public sources


## 🧠 Investigation Approach
1. Identify relevant keywords or signals
2. Locate discussions, posts, or accounts
3. Analyze content and user behavior
4. Cross-check across platforms
5. Identify patterns, inconsistencies, and risks


## 📂 Cases
### Migration Patterns (Vietnam → Australia)
Analysis of Vietnamese online discussions related to migration opportunities, recruitment methods, and potential risk indicators.
Focus:
* Social media recruitment patterns
* Migration-related misinformation
* Behavioral signals across platforms


## 🧩 Key Skills Demonstrated
* OSINT research & analysis
* Pattern recognition
* Critical thinking
* Information verification
* Structured analytical reporting

All investigations are based on publicly available information and conducted for educational purposes only.
